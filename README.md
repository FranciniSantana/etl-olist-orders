# etl-olist-orders
Pipeline de ETL para dados de pedidos do e-commerce brasileiro da Olist.

![Olist Store](/images/olist-logo.jpg)

##  Objetivo
Construir um pipeline simples de ETL utilizando Python e SQL, com foco em boas práticas de Engenharia de Dados.

##  Status do projeto
Em desenvolvimento.
